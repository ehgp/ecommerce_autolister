eCommerce Autolister

Introduction

Autolisting bot that will perform listing functions in both FBMP and Etsy. Ebay is a work in progress

Documentation

Documentation found here

Dependencies

Your command center is located in Dropshipping Items/DROPSHIPPING_SPREADSHEET.xlsx

Must use set_creds.py file with user email and password for code to work and authenticate with the marketplace and webscraper you want

Python 3.8.8 pipenv 2021.11.23

Install

  • Must use set_creds.py file with user email and password for code to work and authenticate with the marketplace and webscraper you want

  • cd to the directory

  • pipenv install

  • pipenv run python fbmp_desktop_sel_list.py

Setup

chromedriver.exe for your chrome version must be installed from here
